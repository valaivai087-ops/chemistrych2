癌症真正想告訴你的事

用白話文理解身體控制系統

CANCER UNDERSTANDING OS｜破除迷思．認識本質．拆解未來治療

❌ 破除迷思｜What Cancer IS NOT

重新認識癌症：
你對癌症的理解可能都是錯的。
癌症不是一張單純的「腫瘤照片」，
而是一套被癌細胞接管的「控制系統」。
❌ 迷思1：癌症是單一基因突變 ▼

真相： 癌症涉及數十條路徑同時失控

📞 比喻：集團詐騙不是只改電話號碼
不是改一個細節，而是整個組織的通訊、財務、人事、供應鏈都重新設計。
癌症也是——基因突變只是開始，接著病毒、訊號、代謝、免疫都跟著改變。

一個腫瘤可能有100種不同的癌細胞亞種，每一種都有自己的突變組合。

❌ 迷思2：癌症是細胞長太快 ▼

真相： 癌症是「活太久」，不是「長太快」

🌱 比喻：雜草vs樹
一株雜草長得快，但活不了三個月。
一棵樹長得慢，但能活100年。
癌細胞就像得到「永生」的種子。它停止自殺訊號，擁有無限複製的能力。

核心能力：「永生複製」(Replicative Immortality) 遠比「快速分裂」更危險。

❌ 迷思3：殺死腫瘤就治好癌症 ▼

真相： 治療本身成為選擇壓力，加速癌症進化

🧬 比喻：殺蟑螂用滅蟲劑
第一次殺死99%的蟑螂。
但活下來的1%對滅蟲劑有抗性。
三個月後，蟑螂回來，而且不怕那種藥了。
癌細胞也是——治療越猛，倖存者越會進化出新的抗藥機制。

真正的未來治療是「控制」，不是「消滅」。

❌ 迷思4：有癌症腫瘤就是癌症 ▼

真相： 許多人身上都有小腫瘤，但沒發病

🏘️ 比喻：有房子 ≠ 有幫派佔據
空房子≠被黑幫佔領的房子。
同樣的，身上有小腫瘤≠癌症爆發。
真正關鍵是：免疫系統有沒有失守？環境有沒有被癌細胞改造？

癌症的發生需要「組織環境允許」、「免疫監控失敗」、「養分充足」三個條件同時滿足。

🎯 癌症四大核心特徵

為什麼同一個癌症會越來越難治？
因為癌細胞一直在學。
它們逐漸習得四種超級能力。

🚪 ① 侵襲性

Invasion

癌細胞打破邊界，從一個地方逃出去。就像有人把圍牆拆掉。

核心機制
分泌酵素MMP拆掉細胞外基質(ECM) → 失去黏附 → 向外侵襲

✈️ ② 轉移性

Metastasis

癌細胞搬家到遠方器官。導致死亡的往往不是原發腫瘤，而是轉移。

核心機制
進入血管 → 逃過免疫 → 循環全身 → 在新地方扎根 → 重新長大

🌳 ③ 異質性

Heterogeneity

一個腫瘤不是「一種癌」。裡面混著100種不同的癌細胞。

核心機制
不同突變 → 不同速度 → 不同代謝 → 不同抗藥 → 無法一種藥全滅

🛡️ ④ 抗藥性

Drug Resistance

治療越多次，癌細胞進化越快。它學會了破解你的藥。

核心機制
基因突變 → 旁路活化 → 增加藥物排出 → 降低細胞凋亡 → 存活下來
🎯 關鍵認知：
這四個能力不是獨立的，而是互相支援、形成閉環。
侵襲 → 轉移 → 新環境裡產生新突變 → 異質性增加 → 抗藥性增加 → 再次侵襲
這就是為什麼癌症一再復發。

⚙️ 癌症控制系統導覽

把癌症看成一個Operating System（作業系統）：
不是一場「戰爭」，而是一個「失控的程式」。
七個控制中心同時失衡，才形成癌症。
① 持續增殖｜Sustained Proliferation ▼

核心問題： 細胞永遠停不下來

正常細胞有上限 → 分裂50次就停止

癌細胞無上限 → 永遠分裂，永不停止

控制路徑： EGFR → RAS → RAF → MEK → ERK → MYC

營養支持： 美寶(EGCG)、EPA+DHA、粒線寶

② 逃避抑制｜Evading Growth Suppressors ▼

核心問題： 把身體的煞車拆掉

正常有P53煞車 → 細胞不會亂長

癌細胞把P53破壞 → 煞車失靈

控制路徑： p53、RB、Cyclin、CDK

營養支持： NAC、維生素C、B群

③ 抵抗凋亡｜Avoiding Apoptosis ▼

核心問題： 該死不死

正常細胞有自殺程式(Apoptosis) → 活該時自己消失

癌細胞關閉自殺程式 → 永遠活著

控制路徑： BCL-2、BAX、Caspase、Survivin

營養支持： CoQ10、維生素E、綠茶多酚

④ 無限複製｜Replicative Immortality ▼

核心問題： 細胞壽命無限延長

正常細胞有端粒(Telomere) → 每分裂一次就短一點 → 最後消失 → 細胞停止複製

癌細胞激活端粒酶(Telomerase) → 每次分裂都自動修復 → 永不衰老

控制路徑： TERT、Telomerase

營養支持： 支持DNA修復與正常細胞衰老機制

⑤ 血管新生｜Angiogenesis ▼

核心問題： 自己蓋水管，運送養分

腫瘤如果沒有血管 → 長不超過1mm → 活不了

癌細胞分泌VEGF → 吸引身體長新血管 → 提供源源不絕的氧氣與養分

控制路徑： VEGF、HIF-1α、FGF

營養支持： 美寶、EPA+DHA、粒線寶、NAC

⑥ 免疫逃脫｜Immune Escape ▼

核心問題： 躲過免疫警察

正常細胞 → NK細胞認識、摧毀

癌細胞穿上隱形斗篷(PD-L1) → 免疫細胞看不到 → 或把免疫細胞變成自己的保鑣

控制路徑： PD-1、PD-L1、CTLA-4、Treg、MDSC

營養支持： 酶菌醣體、孢子益生菌、EPA+DHA、維生素D

⑦ 代謝重編程｜Metabolic Reprogramming ▼

核心問題： 改變發電方式

正常細胞 → 有氧呼吸 → ATP很多 → 能量充足

癌細胞 → 改用糖解作用(Warburg) → 快速產能但效率低 → 但能快速生長與適應缺氧環境

控制路徑： AMPK、mTOR、GLUT1、LDHA

營養支持： 粒線寶、能量方、苦瓜、黃連素、B群

⚡ 系統論領悟：
癌症不是「一個壞」，而是「七個系統同時失控」。
真正的治療不是打一條路徑，而是「同時調理多條路徑」的Network Modulation。
未來精準醫療就是根據每位患者的「特定失控模式」，設計個人化的組合治療。

❓ 常見問題 Q&A

❓ 為什麼同一種癌症，有人治好，有人沒治好？
因為每位患者的癌症DNA、RNA、蛋白質、路徑、微環境都不同。
A病人的肺癌可能由EGFR驅動，B病人的肺癌由ROS1驅動。
同樣的藥對A有效，對B無效，因為他們的癌症「程式碼」不一樣。

未來方向： NGS基因檢測、RNA分析、免疫分型 → 找到「你的癌症」的獨特弱點 → 設計專屬療法
❓ 為什麼化療、標靶、免疫都試過，癌症還是回來？
你殺死了敵人主力軍隊，但沒有消滅敵人母巢（癌幹細胞）。

癌幹細胞的特色：
• 休眠 → 躲過大部分治療
• 自我更新 → 一個變十個
• 高度可塑性 → 環境改變就變身

治療結束後，癌幹細胞甦醒 → 重新長出癌症 → 往往比前一次更抗藥

未來方向： 不只殺癌細胞，還要「關閉癌幹細胞開關」(WNT、NOTCH、YAP抑制劑)
❓ 營養品能治療癌症嗎？
不能。 藥物治療癌症，營養支持正常細胞。

營養品的真正角色：
✓ 提供DNA修復所需原料(B群、葉酸、NAC)
✓ 支持粒線體功能，維持能量代謝
✓ 降低氧化壓力，減少慢性發炎
✓ 支持腸道菌相與免疫監控
✓ 幫助身體從治療副作用復原

比喻： 藥是「醫生做手術」，營養是「家人在旁邊照顧、餵飯、擦汗」。
家人再貼心，也不能代替手術。
但没有家人照顧，患者恢復會很慢。
❓ 如何預防癌症？
記住六個預防支柱：

1️⃣ 高品質睡眠 → 免疫修復、激素平衡
2️⃣ 穩定血糖 → 減少代謝失衡、慢性發炎
3️⃣ 規律運動 → 提高NK細胞、改善循環、降低體脂
4️⃣ 植物營養 → 多酚、類黃酮、植化素，支持抗氧化
5️⃣ 健康腸道 → 70%免疫在腸道，菌相失衡是許多慢性病根源
6️⃣ 壓力管理 → 長期高壓 → 皮質醇升高 → 免疫抑制 → 癌症風險增加

這六項做好，癌症風險下降50%以上。
❓ 為什麼有人長年吸菸沒得癌症，有人不吸菸卻得癌症？
因為癌症不是「一個因素」造成的，而是「多個風險因素累積」。

多因素模式：
吸菸 + 缺乏運動 + 壓力大 + 睡眠差 + 免疫失衡 = 高風險
吸菸 + 運動、睡眠好、管理壓力、免疫強 = 中等風險
不吸菸 + 但缺乏這些健康習慣 = 還是有風險

這就是為什麼有人「看起來危險」卻長壽，有人「看起來健康」卻生病。
真正重要的是整個生活型態，不是單一因素。
❓ 癌症能預防嗎？還是看命？
既不是「完全可預防」，也不是「看命」，而是「風險管理」。

你可以控制的部分(70%)：
• 飲食習慣
• 運動習慣
• 睡眠品質
• 壓力管理
• 戒菸限酒
• 體重管理
• 腸道健康

你控制不了的部分(30%)：
• 遺傳基因
• 環境污染
• 感染病史
• 年齡

把「能控制的70%」做到最好，就已經大幅降低癌症風險了。

🚀 未來癌症治療五大革命

癌症治療正在經歷最大的進化。
從「一種藥殺所有癌症」進化到「每個患者一份客製化治療方案」。
第一代 (1950~1980)
化學治療｜Chemotherapy
核心概念： 殺死快速分裂的細胞
缺點： 好細胞壞細胞一起殺，副作用很大
今天角色： 某些血液癌與胚胎癌仍有效，但多用作為組合治療的一部分
第二代 (1995~2010)
標靶治療｜Target Therapy
核心概念： 關閉癌症的驅動基因(Driver Gene)
代表： EGFR抑制劑、ALK抑制劑、BRAF抑制劑、HER2抑制劑
問題： 癌細胞不斷進化，單一標靶很容易就被抗藥化
今天角色： 仍是主流，但往往需要「多標靶組合治療」
第三代 (2010~現在)
免疫治療｜Immunotherapy
核心概念： 恢復免疫系統辨識、攻擊癌細胞的能力
代表： PD-1/PD-L1抑制劑、CTLA-4抑制劑、CAR-T療法
突破： 有些晚期癌症被完全緩解，效果超越前代治療
問題： 對部分患者無效，需要精準選擇
未來： 從「通用免疫療法」進化到「個人化免疫組合」
第四代 (2020~現在)
可程式化醫療｜Programmable Medicine
核心概念： 不只抑制蛋白質，而是直接修改或摧毀它
新技術：
• PROTAC： 把癌蛋白送進垃圾場降解
• Molecular Glue： 用分子膠把癌蛋白黏到E3酵素，強制降解
• CRISPR基因編輯： 剪掉致癌基因
• Base Editing： 直接修改DNA鹼基
• Prime Editing： 精準插入、刪除、替換DNA
階段： 多項進入臨床試驗，部分已核准上市
第五代 (2024~未來)
AI精準腫瘤學｜AI Precision Oncology
核心概念： AI為每位患者設計最佳個人化治療
技術：
• 多體學分析： 同時分析基因(Genome)、RNA、蛋白質、代謝、微環境、免疫
• Digital Twin(數位分身)： AI建立患者癌症的虛擬模型，預測治療效果
• 個人化疫苗： 根據患者Neoantigen製作專屬mRNA疫苗
• 自適應治療： 根據患者反應動態調整治療策略
階段： 快速發展中，多項臨床試驗進行

最新重點治療方向(2026)

🔬 WNT抑制劑｜對抗癌幹細胞 ▼

作用： 關閉癌幹細胞的「自我更新開關」

為何重要： 癌幹細胞是復發根源，傳統療法往往殺不死它

代表藥物： LGK974、DKN-01(臨床試驗中)

挑戰： WNT對正常腸道、骨骼也重要，副作用控制困難

🔬 PROTAC｜革命性蛋白質降解技術 ▼

作用： 不只阻斷蛋白質，而是把它直接送進垃圾場

優勢： 克服「蛋白質突變後仍活躍」的問題

開發中標靶： KRAS、EGFR、AR、ER、BRD4、BCL6等

多項臨床試驗進行中，未來5年將有多個新藥上市

🔬 個人化癌症疫苗｜neoantigen + mRNA ▼

流程： NGS定序 → 找neoantigen → mRNA製造 → 訓練免疫 → 攻擊癌細胞

優勢： 每位患者疫苗都不同，專一性最高，副作用小

目前： 多項臨床試驗顯示有效，部分已進入後期試驗

挑戰： 製作時間長、成本高、需要與其他療法配合

🔬 自適應治療｜Adaptive Therapy ▼

概念： 不是一直加大藥量，而是「觀察癌症如何改變，動態調整策略」

優勢： 避免把所有敏感癌細胞殺光後，只剩最抗藥的爆炸性增長

目標： 從「治癒」轉為「長期控制」，讓癌症變成慢性病

臨床試驗進行中，初步成果令人鼓舞

🎯 未來五年癌症治療最確定的趨勢：

1️⃣ 從「單一藥物」→ 「多藥物組合」： 同時作用於多條路徑
2️⃣ 從「通用治療」→ 「個人化治療」： AI+多體學分析決定每人的療法
3️⃣ 從「消滅癌症」→ 「控制癌症」： 目標變為長期緩解，而非根治
4️⃣ 從「治療後」→ 「治療中動態調整」： 實時監測，即時改策略
5️⃣ 從「器官分類」→ 「分子分類」： 肺癌、肝癌的分類被「EGFR陽性」「PD-L1高表達」取代
